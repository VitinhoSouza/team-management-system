## System features:
This system aims to manage players and teams, being possible to:
- Create players; 
- Edit Players; 
- Search for players; 
- Exclude players; 
- Create teams; 
- Add players to teams; 
- Remove players to teams; 
- Exclude teams;

## Techs:
- ReactJs;

- Sass;
- Html;
- Redux;
- Firebase (Auth, Realtime Database e Hosting);
- React Router Dom / History (for route redirection);
